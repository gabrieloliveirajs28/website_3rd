Sobre o Ecos do Aço

Um projeto educacional desenvolvido como trabalho acadêmico, explorando história, design e desenvolvimento web.

Objetivo

O que é ?

Este projeto é um guia educacional sobre armas de combate corpo a corpo, desenvolvido como projeto da disciplina de Desenvolvimento Web no 3º semestre do curso de Análise e Desenvolvimento de Sistemas.

O objetivo principal é mostrar um pouco da história, o design e a manufatura de armas brancas de diferentes civilizações — orientais e ocidentais — de forma acessível e visualmente organizada.

Este projeto serviu como oportunidade de aprendizado e aplicar os conceitos de usabilidade estudados em sala, como os princípios do livro "Não Me Faça Pensar" de Steve Krug e os princípios C.A.R.P. de Robin Williams, autora do livro "Design para Quem Não é Designer".

Ferramentas

Tecnologias Utilizadas

O projeto foi construído utilizando puro HTML e CSS, sem uso de frameworks como React ou bibliotecas externas.

HTML5

Estrutura

O esqueleto das páginas foi feito com elementos semânticos do HTML5 como section, aside, nav, header e footer.

CSS3

Estilização

A estilização visual, animações e o menu lateral foram feitos puramente em CSS.

Flexbox

Layout

O sistema de layout das seções, grids de cards e sidebar foram construídos com CSS Flexbox, dispensando o uso de frameworks como Bootstrap.

Checkbox Trick

Interatividade

O menu hambúrguer usa a técnica de CSS "input:checkbox" com o seletor ~ para controlar abertura e fechamento.
